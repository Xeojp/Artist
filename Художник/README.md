# Виртуальный художник

Python-приложение для генерации картин в стиле разных художников с помощью Stable Diffusion.

## Возможности

* Генерация картин по текстовому описанию.
* Выбор стиля художника (Van Gogh, Picasso, Monet, абстрактный, киберпанк).
* Настройка яркости, контраста и резкости.
* Просмотр галереи всех сгенерированных картин.
* Запуск через веб-интерфейс или командную строку.

## Установка

1. Клонируйте репозиторий:

```bash
git clone <репозиторий>
cd virtual_artist
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Скачайте модель Stable Diffusion локально (опционально):

```bash
python app/download_models.py
```

## Запуск

### Веб-интерфейс

```bash
python web_ui/interface.py
```

* Ввод текста, выбор стиля, настройка яркости/контраста/резкости.
* Галерея всех картин обновляется автоматически.

### CLI

```bash
python app/main.py --prompt "ночной город" --style vangogh --brightness 1.2 --contrast 1.1 --sharpness 1.0
```
